2019年春招实习经验

2019-04-23

🤦‍♀️这篇review大概四月份就写了，现在十一月十号了，终于想起来要把它完善了。

春招大概是18年的圣诞节就开始了，此处@上海字节EE团队，感谢他们的南大圣诞节专场。

2108.12.25 圣诞节。上海字节EE团队在nju软院费楼开了专场宣讲会，介绍了下部门的职责，学姐分享了一下她的实习体验。然后开始面试，三道算法题，一道动态规划，一道智力题，一道数学题。动态规划题目 类似于最长能看的电影场数。

以下为四月份的原文。

细数一下，春天以来 做过 微软一次笔试，三次面试。阿里 三次面试 腾讯鸽了提前批笔试 正式批笔试 美团笔试放弃了 华为一次笔试 二次面试 招商银行 笔试

心力憔悴 最主要的是感觉自己面试过程中没啥提升 也没有啥准备。 先等offer通知吧，回来补

春招总结 续 2019.04.25

楼主背景 华五某校 软件学院 无大型上线项目 无竞赛经历 无获奖经历 成绩中流 整个春招大概持续一个半月 3.11-4.25 预计会收到一两个offer(大胆奶一口)

蚂蚁金服 Java 研发 一面凉 被捞 测试开发岗 一面 过 二面凉

腾讯 笔试时间冲突 鸽了

华为 笔试 460 一面 二面 聊聊天 现在录用排序中 估计没啥问题

微软 SWE（STCA）笔试 a了第一道 一面 二面 三面 等offer中

头条 寒假 提前拿到了ios岗实习offer 就没再投

美团 笔试凉

招行信用卡中心 笔试凉

自己的心路历程，因为寒假拿到了头条的offer，所以春招就心态放松了，一直吃老底，没有补充新的知识，面完也没有及时复习，所以很多知识点都没有记住，比如Spring的事物管理机制，消息队列的实现、算法，如何加快大数据表的查询等等，所以很多凉了也正常。一直面一直

可以提前做好的准备，比如 项目开发中遇到什么难点，是怎么解决的， 未来的职业规划呀（千万不要说干几年跳槽的蠢话 哭泣）读研还是工作（建议回答工作）这种没有标答的题目

如果不想背各种知识点 建议刷一些leetcode 然后投 微软 等外企 或者华为 相对来说水一点（可能存在幸存者偏差）比如面微软 你不需要知道 一个url输入浏览器后都经历了什么，也不需要知道spring IOC，AOP，事务管理，也不需要知道如何查询一个超大数据量的表，也不需要知道JVM的组成，垃圾回收，也不需要管各种高并发多线程悲观锁乐观锁。

前路漫漫 且行且珍惜 祝大家都能拿到理想的offer。 先上岸啦！
