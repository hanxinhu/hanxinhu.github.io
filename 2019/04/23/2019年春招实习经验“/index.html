2019年春招实习经验

2019-04-23

🤦‍♀️这篇review大概四月份就写了，现在十一月十号了，终于想起来要把它完善了。

嗯现在十一月十六号了。 OK 下面是正文。

春招大概是18年的圣诞节就开始了，此处@上海字节EE团队，感谢他们的南大圣诞节专场。

2108.12.25 圣诞节。上海字节EE团队在nju软院费楼开了专场宣讲会，介绍了下部门做的事情，一个15级学姐分享了一下她的实习体验。然后开始笔试，三道算法题，一道动态规划，一道智力题，一道数学题。动态规划题目 类似于最长能看的电影场数。我大概做出来一道半，很不幸的没有过，然后看着室友们收到面试通知，呜呜呜，但是我并没有放弃，在现场笔试的同时，又开放了三场线上的笔试，我猜想现场的笔试应该没有记录在系统中，那么我多参加一次笔试应该也可以，于是就报名了大概一月十二号的最后一场笔试，也是两个小时三道算法题，一道easy 两道meduim 笔试链接以及后面我的题解，菜鸡的我依然只做出来一道easy，medium有思路但是并没有做出来，最后20分收场，出来后就觉得肯定没戏了，于是和朋友出去喝酒约饭了，没想到在咖啡店门口收到了笔试通过的消息，什么叫惊喜，这就是惊喜。然后想着我室友们都没有过那么我应该几率也不是很大吧，就依然纵情狂欢。快乐1912。回去后就认真问了室友都被问到了什么问题，答曰，多线程，java虚拟机，生命周期，垃圾回收，TCD，DCP，不一而足，还有经典的手写红黑树！（x）但是两道算法题必不可少。还有单例模式。一面挺顺利的，虽然算法题给小姐姐解释了半天，二面也比较顺利，虽然他质疑了一下我的单例模式。还有经典的用两个栈实现一个队列的功能，要支持多线程。面试完感觉还行，多半应该是稳了。然后过了两个多小时就收到了HR的消息，问我投的安卓岗位已经差不多了，但是还可以进，ios岗位空缺还比较多，要不要考虑一下。我心想IOS和安卓也差不多，就接下了第一个offer。美滋滋，字节是我很喜欢的一个公司，然后就欢欢喜喜过大年，年后到学校就开始春招啦。

首先院群里各种各样的内推开始轰炸了，最多的是阿里系，估计是有指标要完成吧。当时觉得阿里蚂蚁金服最牛，就等到了一个蚂蚁的内推投递， 然后过了好久没有消息，一天晚上大概八点半打来了电话开始面试，cold call,出乎意料， 很侧重项目经历，问了好多做的项目都内容，而且借机问了对高并发之类的处理。还是挺有意思的。 没想到后端开发还是一面挂。。过了不久问我要不要去测试，就开始面测试，一面过了，二面是先做题，做题后再给他讲，最后悔的是问我人生规划，我说我干几年然后跳出去，想当一个ios 个人开发者，拥有睡后收入 大概违背了阿里的价值观，然后就杳无音讯了。再也没有被捞起来。腾讯当时因为要填的东西太多 了，一不小心错过了提前批，呜呜，错过了免笔试的机会，一定要投提前批，因为正式批很可能没有HC了，正式批的时候，因为笔试安排在了清明节，当时碰巧在外面玩，就放弃了，大概和腾讯没有缘分。 然后跟风投了个华为保底，先笔试，相对比较简单，满分五百做出来460，然后就面试了，一面很顺利，就聊聊天做了一道题，中午被安排去食堂吃个饭，没想到刚吃完就被安排上了二面，是个老大爷，客套两局，问了问，我是怎么学习的，怎么带领一个团队做一个项目的，怎么分工的 问我能不能接受华为的工作强度，当然可以了。然后就结束了。 后来也稳稳的收到了offer。 也跟风投了微软，大概四月份吧，因为没想着会过，所以做的比较随意，微软笔试和阿里测试二面还撞车了，匆匆做了一道题就交上去了。然后收到了已经招满的邮件，问我对其他岗位有没有意向，正有点失落时，收到另外一封说上一封是系统错误，面试还在安排中。 大起大落，人生就是充满惊喜。然后买票准备去苏州面试，14:00开始，我大概12点就到了吧，在那边等了一段时间，然后开始面试，一面是一个senior 的小姐姐，出来一道leetcode的原题，两数字之和，然后引申到了三数之和，二面是个小哥，software II ，也是我后来的mentor，可惜并没有认出来我，还让我用英语自我介绍了一下，还好早有准备，在高铁上认真背了一路。三面是个在家办公的大老板manager, cozy chen, 出了一道 中文字符串转换成数字的题目，面经在链接中 微软面经。 基本上都做出来了，虽然也是满头大汗，然后就稳了，因为比较想去的公司都差不多了，后面招商信用卡美团的笔试都没有耐心做了，就等offer了。后面也如愿收到了华为，微软的offer。然后一番思考。想着实习主要是去体验感受氛围的，也不是完全为了挣钱，又看到我院大佬放弃的蚂蚁金服的offer选择 了微软，我也就跟风选择了微软，享受了一段955的快乐生活。还有一个原因是，其他公司可能还有日常实习生，后面还有机会，不着急。

微软实习经历看另外一篇。

写的时候好多朋友都求秋招了，看到他们拿了sp,ssp还是挺酸的，而我还要继续上学，希望两年后我也能如愿以偿吧，要继续多参加比赛多拿奖多刷题。

以下为四月份的原文。

细数一下，春天以来 做过 微软一次笔试，三次面试。阿里 三次面试 腾讯鸽了提前批笔试 正式批笔试 美团笔试放弃了 华为一次笔试 二次面试 招商银行 笔试

心力憔悴 最主要的是感觉自己面试过程中没啥提升 也没有啥准备。 先等offer通知吧，回来补

春招总结 续 2019.04.25

楼主背景 华五某校 软件学院 无大型上线项目 无竞赛经历 无获奖经历 成绩中流 整个春招大概持续一个半月 3.11-4.25 预计会收到一两个offer(大胆奶一口)

蚂蚁金服 Java 研发 一面凉 被捞 测试开发岗 一面 过 二面凉

腾讯 笔试时间冲突 鸽了

华为 笔试 460 一面 二面 聊聊天 现在录用排序中 估计没啥问题

微软 SWE（STCA）笔试 a了第一道 一面 二面 三面 等offer中

头条 寒假 提前拿到了ios岗实习offer 就没再投

美团 笔试凉

招行信用卡中心 笔试凉

自己的心路历程，因为寒假拿到了头条的offer，所以春招就心态放松了，一直吃老底，没有补充新的知识，面完也没有及时复习，所以很多知识点都没有记住，比如Spring的事物管理机制，消息队列的实现、算法，如何加快大数据表的查询等等，所以很多凉了也正常。一直面一直

可以提前做好的准备，比如 项目开发中遇到什么难点，是怎么解决的， 未来的职业规划呀（千万不要说干几年跳槽的蠢话 哭泣）读研还是工作（建议回答工作）这种没有标答的题目

如果不想背各种知识点 建议刷一些leetcode 然后投 微软 等外企 或者华为 相对来说水一点（可能存在幸存者偏差）比如面微软 你不需要知道 一个url输入浏览器后都经历了什么，也不需要知道spring IOC，AOP，事务管理，也不需要知道如何查询一个超大数据量的表，也不需要知道JVM的组成，垃圾回收，也不需要管各种高并发多线程悲观锁乐观锁。

前路漫漫 且行且珍惜 祝大家都能拿到理想的offer。 先上岸啦！
